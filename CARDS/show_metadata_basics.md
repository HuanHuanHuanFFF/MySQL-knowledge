# 元信息查看指令

**Point**: 常用命令：`SHOW DATABASES;`、`SHOW TABLES;`、`DESC tbl;`、`SHOW CREATE TABLE tbl;`。

**Why**: 排查结构、校验 DDL 结果或查看默认字符集、索引定义。
